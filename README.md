# FileEditor

Проект служит для изучения языка C. В процессе чтения учебника необходимы также и практические навыки написания кода, что этот проект и позволил получить

### Функции
  1. Выбор файла
  2. Просмотр содержимого файла
  3. Добавление строк в файл
  4. Удаление строк из файла
  5. Изменение строк файла
  6. Сохранение файла

Все функции работают, также был написан Makefile для удобной сборки
